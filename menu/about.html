Project-3 Micro Frontend & Micro Service Developer

Luaran Proyek 3 Informatika adalah membangun sebuah aplikasi yang harus terdiri dari frontend dan backend.Dengan tetentuan:

1. Frontend wajib di deploy di github pages dengan menggunakan konsep serverless micro frontendmenggunakan Javascript ES6+

2. Backend minimal menggunakan RESTFull,boleh menggunakan web socket atau graphql untuk berinteraksi dengan frontend

3. Backend wajib di hosting di GCP (Cloud Function/ Clound Run),AWS,Heroku,VPS,Hosting

4. Backend wajib menggunakan sistem Aunhentikasi berbasiskan token PASETO V4

5. Direkomendasikan membangun backend menggunakan Go Package atau Framework Go Fiber

6. Semua proses Proyek 3 menggunakan metode belajar Pomodoro dengan aplikasi pomokit

HealHero: Aplikasi Informasi Kesehatan

Aplikasi ini dirancang untuk membantu individu dalam menjaga kesehatan mereka dan mengatasi berbagai masalah kesehatan.

Fitur Utama HealHero :

1. Penyuluhan Kesehatan : HealHero menyediakan informasi kesehatan yang akurat dan dapat dipercaya kepada pengguna. Artikel,video, dan sumber daya laiinya membantu pengguna untuk memahami penyakit, kondisi dan cara menjaganya.

2. Konsultasi Bersama Dokter : HealHero menyediakan layanan untuk berkomunikasi bersama dokter yang sudah ada pilih sesuai dengan jenis

3. Pemesanan Obat secara langsung : HealHero menyediakan layanan pemesanan obata secara langsung yang dilakukan oleh pembeli

Manfaat Pengguna HealHero

1. Kesadaran Kesehatan: Aplikasi ini membantu meningkatkan kesadaran tentang kesehatan pribadi. Dengan memantau data kesehatan mereka, pengguna lebih cenderung untuk memperhatikan perubahan dan mencari perawatan jika diperlukan.

2. Pencegahan Penyakit: Dengan penyuluhan kesehatan yang mudah diakses, pengguna dapat memahami langkah-langkah pencegahan yang perlu mereka ambil untuk menghindari penyakit tertentu.

3. Kemudahan Manajemen Kesehatan: Fitur pengingat obat dan jadwal dokter membantu pengguna dalam manajemen kesehatan mereka dengan lebih baik, terutama bagi mereka yang memiliki penyakit kronis.

4. Peningkatan Gaya Hidup: Melalui pelacakan aktivitas fisik, pengguna diberi insentif untuk hidup lebih aktif dan sehat.
